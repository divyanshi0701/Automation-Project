# Playwright Automation Framework

## 🚀 Overview
This project is an end-to-end test automation framework built using Playwright with JavaScript.

## 🧰 Tech Stack
- Playwright
- JavaScript
- Node.js
- GitHub Actions (CI/CD)

## ✅ Features
- Page Object Model (POM)
- UI + API Testing
- Parallel Execution
- Retry Mechanism
- HTML Reports
- CI/CD Integration

## 🧪 Test Coverage
- Login functionality
- Add to Cart
- Checkout flow
- API validation

## ▶️ Run Tests
npm install
npx playwright test

## 📊 Reports
npx playwright show-report
